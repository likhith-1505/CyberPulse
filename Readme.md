# CyberPulse

A security analysis platform I built to scan URLs, files, emails, and network captures. Think of it as a mini SOC tool you can run locally.

Still a work in progress — URL analyzer and file scanner are done, rest is coming.

---

## What works right now

- **URL Analyzer** — checks domain age, WHOIS, DNS, suspicious keywords and TLDs. Gives a risk score.
- **File Scanner** — uploads to VirusTotal and shows you which engines flagged it and why.
- Email Analyzer — coming soon
- PCAP Analyzer — coming soon

---

## Stack

- Frontend: React + TypeScript + Vite + Tailwind
- Backend: FastAPI + Python 3

---

## Requirements

- Python 3.10+
- Node.js 18+
- A free VirusTotal API key (get one at virustotal.com — takes 2 minutes)

## Setup

### 1. Clone it
```bash
git clone https://github.com/likhith-1505/CyberPulse.git
cd CyberPulse
```

### 2. Backend
```bash
cd Backend
python3 -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

Now create a `.env` file in the Backend folder:
VIRUSTOTAL_API_KEY=paste_your_key_here

Don't have a key? Go to virustotal.com, make a free account, then go to your profile → API Key. Free tier gives you 500 requests/day which is plenty.

Start the backend:
```bash
uvicorn main:app --reload --port 8000
```

### 3. Frontend

Open a second terminal:
```bash
cd Frontend
npm install
npm run dev
```

### 4. Open the app
http://localhost:5173

Backend API docs (auto-generated by FastAPI):
http://localhost:8000/docs

## Common issues

**npm not found** — Node isn't installed or nvm isn't loaded. Run `source ~/.bashrc` and try again.

**scan failed / API key error** — check your `.env` file exists in the Backend folder and actually has your key in it. File should look like `VIRUSTOTAL_API_KEY=abc123...` with no quotes.

**CORS error in browser** — make sure the backend is running on port 8000. The frontend expects it there.

**venv not activating on Windows** — run `Set-ExecutionPolicy Unrestricted -Scope CurrentUser` in PowerShell first.
